# Base Vagrantfile for Rails developers with Chef

This vagrantfile will setup for you Ubuntu 14.04 virtual machine with VirtualBox and will provise it with Chef.

### Provision include:
- apt-get update;
- build-essential (C compiler);
- ruby_build;
- nodejs;
- git;
- vim;
- rmv;
- zsh;
- oh-my-zsh.
